================================================================================
                            PROJECT DESCRIPTION
================================================================================

FriendFinder is a modern dating application built with React and Vite. It 
features a mobile-first design that mimics native mobile app experiences. The 
app includes comprehensive user authentication, profile management, matching 
system, messaging, search functionality, and safety features including profile 
verification and photo privacy controls.

================================================================================
                          SAFETY FEATURES
================================================================================

1. PROFILE VERIFICATION
   - Users must complete all signup steps to be verified
   - Verified status stored in user object
   - Only verified profiles shown to verified users

2. PHOTO PRIVACY
   - Photos hidden by default (placeholder shown)
   - Photos only visible after clicking profile
   - Only first photo shown in preview
   - No photos shown for unverified profiles

3. PROFILE PREVIEW
   - Click to view full profile
   - Modal preview before interaction
   - Complete profile information
   - Action buttons (Like/Pass)

4. USER BLOCKING
   - Block users functionality
   - Blocked users filtered from results
   - Report user option

5. VERIFIED-ONLY DISPLAY
   - Verified users only see verified profiles
   - Unverified users see limited content
   - Safety-first approach

================================================================================
                          DATA FLOW
================================================================================

1. USER SIGNUP FLOW:
   Verification → Basic Info → Bio/Photo → Interests → Details → Complete
   → User created → Verified = true → Saved to localStorage

2. LOGIN FLOW:
   Enter credentials → Verify OTP/Password → Get user → Save to localStorage
   → Set authenticated → Show main app

3. MATCHING FLOW:
   View profile → Swipe right → Like saved → Check for match → 
   If match → Save match → Show animation → Add to matches

4. SEARCH FLOW:
   Enter query → Apply filters → Filter profiles → Display results → 
   Click profile → Show preview → View details

5. MESSAGING FLOW:
   Select match → Open chat → View messages → Send message → 
   Update last message → Show unread count
